# ⏭ Auto YouTube Shorts Scroller

Automatically scrolls to the next YouTube Short when one ends.  
Includes a sleek floating control panel with play/pause, delay settings, and skip controls.

## ✨ Features

- Auto-scrolls when a Short ends
- Configurable scroll delay (default: 1500ms)
- Skip threshold — skip after watching X% of a video
- Loop guard to prevent re-scrolling the same Short
- Floating control panel with live progress bar

## 📦 Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) browser extension
2. Click the link below to install the script directly:

👉 **[Install Script](https://raw.githubusercontent.com/SecuredPixels/auto-youtube-shorts-scroller/main/auto-youtube-shorts-scroller.user.js)**

> Tampermonkey will detect the `.user.js` file and prompt you to install it automatically.

## 🛠 Usage

- Navigate to any [YouTube Short](https://www.youtube.com/shorts/)
- Click the **⏭** floating button (bottom-right) to open the control panel
- Toggle auto-scroll on/off, adjust delay, set a skip threshold

## ⚙️ Settings

| Setting | Default | Description |
|---|---|---|
| Auto-scroll | On | Scroll when video ends |
| Scroll delay | 1500ms | Wait time before scrolling |
| Skip threshold | 0% | Skip after watching X% (0 = end only) |
| Loop guard | On | Prevents re-scrolling same Short |
